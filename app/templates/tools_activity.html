Credits
Vigilant doesn't archive EVE's full player-count history itself — the data shown here predating this dashboard's deployment is sourced from two community projects that have been doing the patient work for over two decades:
  • Chribba / OMG Labs — eve-offline.net — concurrent player count tracking since 2003-05-28, the canonical historical source.
  • Adminor.net — eve-offline.com — modern rebuild of the same dataset with a clean JSON API and CCP-newborns overlay.
Both sites serve their data freely. We pull the historical backfill once at first startup, then go silent — live data going forward comes directly from CCP's ESI /status/ endpoint. If you find this view useful, consider visiting their sites directly — community-run infrastructure runs on goodwill.
