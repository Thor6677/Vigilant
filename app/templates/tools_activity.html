Credits
Vigilant doesn't archive EVE's history itself — the long-tail data shown here predating this dashboard's deployment is sourced from community projects that have been doing the patient work for nearly two decades:
  • Chribba / OMG Labs — eve-offline.net — concurrent player count tracking since 2003-05-28, the canonical historical PCU source.
  • Adminor.net — eve-offline.com — modern rebuild of the player-count dataset with a clean JSON API and CCP-newborns overlay.
  • zKillboard — daily kill-count totals (/api/history/totals.json) since 2007-12-05. Provides the long-tail daily-kills series.
These sites serve their data freely. Vigilant pulls each historical archive once at first startup, then goes silent — live data going forward comes directly from CCP's ESI /status/ endpoint and the killmail.stream consumer. If you find this view useful, consider visiting their sites directly — community-run infrastructure runs on goodwill.
